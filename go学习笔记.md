# VScode安装

```
ctrl + shift + P
输入 go:install/update tools 安装go相关的提示包

设置自动保存

设置snippets
{
    "println":{
    "prefix": "pln",
    "body":"fmt.Println($0)",
    "description": "println"
    },
    "printf":{
    "prefix": "plf",
    "body": "fmt.Printf(\"$0\")",
    "description": "printf"
    }
}


```
